# RAJAK Frontend

React + Vite frontend for auth, rides and bookings.

## What’s new

- Uber-like, map-first booking and hosting flow
- Modern navigation with quick CTA to book or host
- Refreshed auth screens and cards that match the new visual system

## Run

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open the printed local URL (typically http://localhost:5173)

## API

Set `VITE_API_BASE_URL` if backend is not at `http://127.0.0.1:8000/api/v1`.
